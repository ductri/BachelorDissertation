How to run
- set base_directory = "path_to_this_directory"
- run in command line: python main.py

Description:
- main.py: file to run, extract features and call functions in other files
- preprocessing.py: file to preprocess data
- so-cal.py: file to extract so-cal feature
- negation.py: file to extract negation feature
- metamap_output.txt: output after data run through metamap
- metamap*.jar: different versions metamap, choose version to run in proprecessing.py file
